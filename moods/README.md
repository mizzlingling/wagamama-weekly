# 金剛心情圖片

這個資料夾存放金剛不同心情的圖片，每天會根據運勢自動切換。

## 需要的圖片

請準備 7 張不同表情的金剛圖片，命名如下：

1. **happy.png** - 開心表情
   - 用於：開啟新可能性的日子
   - 特徵：明亮、樂觀、充滿希望

2. **communicative.png** - 溝通表情  
   - 用於：人際互動、交流的日子
   - 特徵：友善、開放、善於表達

3. **energetic.png** - 活力表情
   - 用於：改變、行動、突破的日子
   - 特徵：充滿活力、積極、勇敢

4. **introspective.png** - 沉思表情
   - 用於：獨處、反思、內省的日子
   - 特徵：平靜、深思、內斂

5. **creative.png** - 創意表情
   - 用於：創造力巔峰的日子
   - 特徵：靈感、藝術感、想像力

6. **stable.png** - 穩定表情
   - 用於：處理實際事務的日子
   - 特徵：踏實、可靠、專注

7. **calm.png** - 平靜表情（備用）
   - 用於：平和、寧靜的日子
   - 特徵：安詳、放鬆、和諧

## 圖片規格

- **格式**: PNG（支持透明背景）
- **尺寸**: 建議 400x400px 或更高
- **背景**: 透明背景最佳
- **風格**: 卡通、可愛、富有表情

## 如何生成圖片

### 方案 A：AI 生成
使用 DALL-E、Midjourney 或 Stable Diffusion 生成：

提示詞範例：
```
A cute cartoon gorilla/King Kong character with [happy/thoughtful/energetic] expression,
simple design, friendly face, transparent background, emoji style
```

### 方案 B：手繪或設計
使用 Figma、Illustrator 等工具創作不同表情的金剛。

### 方案 C：現有圖片修改
找到基礎金剛圖片，用圖片編輯軟體調整表情。

## 當前狀態

目前所有圖片都使用相同的 kingkong.png 作為佔位符。
請替換成真實的不同表情圖片以啟用完整效果。
